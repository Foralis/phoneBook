Практика: phone_book.sh

Описание: Создать скрипт на Bash "Телефонная книга".

Необходимая функциональность:

Добавление новой записи с фамилией, именем, телефоном и заметками.
Удаление записи.
Редактирование существующих записей.
Вывод всех записей в телефонной книге.
Поиск записей по фамилии или имени.

При запуске скрипта добавить выбор файла телефонной книги.
Данные хранить в файле любого формата (никаких БД).
Пользовательский интерфейс - меню в командной строке (например, bash select).
